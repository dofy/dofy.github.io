关于 Homebrew

Homebrew 是 macOS 下最常用、最强大的软件包管理器，每一台 macOS 系统都应该安装一份。

  • Homebrew 官方网站 - https://brew.sh/
  • GitHub 项目地址 - https://github.com/SpaceVim/SpaceVim
